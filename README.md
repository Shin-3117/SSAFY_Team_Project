### 투자 추천 알고리즘

이 알고리즘은 금, 석유, 주가지수(KOSPI, KOSDAQ, KRX, 테마)와 같은 다양한 금융 자산에 대한 시장 데이터를 분석하여, 현재 시장의 추세를 파악하고 미래 가격을 예측합니다.

### 작동 원리

1. **데이터 수집 및 전처리**:
   - 금융 시장 데이터베이스에서 최신 데이터를 추출합니다.
   - 사용자가 요청한 기간(`cnt`)에 해당하는 데이터를 선택하여 분석합니다. 예를 들어, 사용자가 30일을 선택하면 최근 30일 동안의 시장 데이터를 사용합니다.
   - 각 데이터 포인트는 날짜와 해당 날짜의 금융 자산 가격을 포함합니다.
2. **선형 회귀 분석**:
   - 시장 데이터에 대해 선형 회귀 모델을 생성하고 훈련합니다.
   - 날짜 데이터는 `toordinal` 메소드를 사용하여 숫자로 변환되며, 이는 모델이 날짜 데이터를 처리할 수 있도록 합니다.
   - 모델은 날짜와 가격 간의 선형 관계를 찾아내어, 시장의 기본 추세를 결정합니다.
3. **추세 지표 및 미래 가격 예측**:
   - 모델의 기울기는 시장의 현재 추세를 나타내는 지표로 사용됩니다. 기울기가 양수면 상승 추세, 음수면 하락 추세를 의미합니다.
   - 또한, 모델은 주어진 미래 날짜(예: 현재로부터 3일 후)에 대한 가격 예측을 수행합니다. 이는 `timedelta`를 사용하여 현재 날짜에서 원하는 일수만큼 더한 후, 해당 날짜에 대한 예측을 진행합니다.
4. **결과 제공**:
   - 알고리즘은 현재 시장 추세와 미래 예측 가격을 사용자에게 제공합니다. 이 정보는 투자 결정을 내리는 데 중요한 통찰을 제공할 수 있습니다.
